# Homelab GitOps

Plateforme DevOps Kubernetes pour homelab.

## Objectif

- Kubernetes avec kubeadm
- GitOps avec Argo CD
- Ingress avec NGINX Ingress Controller
- Exposition publique via NGINX Proxy Manager
- Accès admin via WireGuard
- Monitoring avec Prometheus/Grafana
- Logs avec Loki
- Stockage avec Longhorn
- Backup avec Velero
